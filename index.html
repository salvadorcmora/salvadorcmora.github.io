2) GT Movies Store — Description

Overview: A movie catalog web app that lets users browse titles, view details, and simulate rent/purchase actions.

  • Homepage: welcoming header and navigation.
  • Catalog: grid/list of movies with search/filter (title, genre, year).
  • Details: synopsis, rating, release year, and CTA.
  • Checkout: simple cart flow to demonstrate the purchase path.

User stories are addressed by enabling discoverability (browse/search), clear detail views, and an end-to-end purchase simulation.

Live demo: Open GT Movies Store

3) Process Description

  1. Planning: mapped screens to user stories and wrote a checklist.
  2. Design: wireframed layout and navigation; chose a simple style guide.
  3. Implementation: built incrementally (home → catalog → details → checkout).
  4. Testing & Debugging: verified each feature before moving on; used console/devtools and peer feedback.
  5. Refinement: polished responsiveness and accessibility, cleaned code and CSS.
